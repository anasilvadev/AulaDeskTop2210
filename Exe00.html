Métodos Dom

Os métodos permitem manipular elementos do documento HTML com JS

  1. document.getElementById("nome_do_id") - pega o elemento do html via Id.
  2. innnerHTML - inserir/alterar conteudo.
  3. mudar o estilo - Alteração de css via js.
  4. classlist - insere uma class no elemento erm questão
    • .add - adiciona uma class no elemento.
    • .remove - remove uma class no elemento.
    • .toggle - alterna entre inserir e remover a class.
